1. Building on Win32

1.1 Prerequisites:

Windows XP/Vista/7
Visual Studio 2008 Standard Edition or above with C++ and C# installed
Display card which supports OpenGL 2.1 or above

1.2 Common building steps
Append the MCore3D's 'Bin' directory to the 'Path' environment variable

1.2 Building the testes
Open 'Test/Test.sln'; and simply build and run the test projects.
